# PITCHES

This website is used to get pitches from across the world and from different broadcasting channel 

## Technologies Used

-Html
-css
-Python


##### Setup Instructions and Installation

- Clone this repository to a location in your file system. `git clonehttps://github.com/ChelRono/Pitch-project.git`
- Open terminal command line then navigate to the root folder of the application. `cd P`
- Open `http://127.0.0.1:5000 ` on your Browser.

## Known Bugs

If you find a bug (the website couldn't handle the query and or gave undesired results), kindly open an issue here by including your search query and the expected result.

If you'd like to request a new function, feel free to do so by opening an issue here. Please include sample queries and their corresponding results.
